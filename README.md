# Money Management System
The Money Management System is a web application designed to help users track and manage their financial transactions efficiently. It allows users to record incoming and outgoing amounts, view totals, and organize their financial data in a simple and user-friendly interface.
This application helps users monitor their daily financial activity, maintain better control over their expenses and income, and keep their records organized in one place.
## Features
- Add new financial transactions (income or expense)
- Record transaction amount, reason, and date
- View total income and total expenses
- Display overall balance
- Edit or delete individual transactions
- Delete all transactions at once
- Export data to Excel file
- Input validation with user feedback
